# 02 - 机器学习基础

> 机器学习是AI的核心方法论。作为PM，你需要理解不同算法的适用场景和边界。

## 📚 本章节学习目标

- [ ] 理解监督学习、无监督学习、强化学习的区别
- [ ] 掌握经典ML算法的原理和适用场景
- [ ] 理解特征工程和模型评估方法
- [ ] 能判断什么问题适合用什么算法

## 🎯 核心知识点

### 2.1 机器学习范式
- 监督学习：分类 vs 回归
- 无监督学习：聚类、降维、生成模型
- 强化学习：MDP、策略、值函数
- **PM视角**: 不同业务场景对应不同学习范式

### 2.2 经典算法
- 线性/逻辑回归
- 决策树与随机森林
- 支持向量机(SVM)
- K近邻(KNN)
- K-Means聚类
- **PM视角**: 理解算法复杂度、数据需求、可解释性

### 2.3 特征工程
- 特征选择方法
- 特征缩放与编码
- 降维技术
- **PM视角**: 数据质量直接影响产品效果

### 2.4 模型评估
- 准确率、精确率、召回率、F1
- ROC曲线与AUC
- 交叉验证
- 偏差-方差权衡
- **PM视角**: 如何选择业务指标？如何处理数据不平衡？

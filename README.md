## 🎯 知识库使用哲学

### 你不是要成为研究员

你的目标不是推导公式、写代码、设计芯片，而是：

```
❌ 不需要：推导Transformer的数学公式
✅ 需要：知道Transformer为什么适合长序列，RNN为什么不行

❌ 不需要：手写CUDA kernel
✅ 需要：知道GPU利用率低可能是因为Memory Bound

❌ 不需要：设计NPU架构
✅ 需要：知道什么时候选GPU、什么时候选NPU、什么时候选FPGA

❌ 不需要：训练大模型
✅ 需要：知道微调、RAG、Prompt Engineering的适用场景和成本
```

# 01 - AI基础与数学

> 理解AI的数学基础是成为优秀AI产品经理的第一步。
> 你不需要成为数学家，但需要理解AI模型背后的数学直觉。

## 📚 本章节学习目标

- [ ] 理解向量、矩阵、张量的基本运算
- [ ] 掌握概率论在AI中的应用（贝叶斯、MLE）
- [ ] 理解梯度下降和优化原理
- [ ] 能用Python实现基础数据处理和可视化

## 🎯 核心知识点

### 1.1 线性代数
- 向量空间与矩阵运算
- 特征值分解与奇异值分解(SVD)
- 主成分分析(PCA)原理
- **PM视角**: 理解降维、嵌入空间的直觉

### 1.2 概率论与统计
- 概率分布（正态、伯努利、多项式）
- 贝叶斯推断与最大似然估计
- 信息论基础（熵、KL散度、互信息）
- **PM视角**: 理解模型不确定性、A/B测试统计基础

### 1.3 微积分与优化
- 偏导数与梯度
- 凸优化基础
- 拉格朗日乘子法
- **PM视角**: 理解模型训练为什么能收敛

### 1.4 Python编程基础
- NumPy数组操作
- Pandas数据处理
- Matplotlib可视化
- **PM视角**: 能独立跑通Demo，理解数据流程
